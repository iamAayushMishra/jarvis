# Commands Syntax description
1 Kick >kick <member> Kicks a member from the server
2 Ban Ban (@member) [Reason] Ban a user from the server
3 Mute mute (@member) [Reason] Mute a user from the server
4 Warn >warn (@member) [Reason] Warn's a user
5 Whois >whois (@member) Sends info about provided user
6 Random Password Generator >rdp (@username) [length of pass] Generate a strong Random password for Your personal use
7 Vaccine >vaccine [pincode] Tells about available vaccine slots in the given area
8 Weather >weather [city name]/[pincode] Gives weather forecast of the given area
9 Dictionary >dictionary [word] Searches for the word in dictionary
10 Simp >simp [@user] Find the Simp user of the server
